MVC File Structure Explained

Take a look at the following file structure.

    Main App

    server.js

    This file is the initial starting point for the Node/Express server.

    schema.sql

    This file is intended to help developers get their SQL Databases setup correctly.

    app

    routes

    api-routes.js

    api-routes.js - this file offers a set of routes for displaying and saving data to the db

    html-routes.js

    this file offers a set of routes for sending users to the various html pages

    configuration

    connection.js

    connection.js - this file offers a set of routes for displaying and saving data to the db

    orm.js

    orm.js - this file offers a set of routes for sending users to the various html pages

    public

    css

    images

    views

    intro-to-mvc.html

    structure.html

    expansion.html

    js

    intro-to-mvc.js

    code here handles the logic and queries fo the database related to intro-to-mvc.html

    structure.js

    code here handles general logic and queries for the database related to structure.html

    expansion.js

    code here handles queries fo the database related to expansion.html
