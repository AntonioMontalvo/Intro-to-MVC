Object Relational Mapping

O.R.M Definition

Object Relational Mapping is a technique to convert datat between two different systems using object- oriented programming. Because SQL (DBMS) only stores and manipulates scalar values (integeres, strings in tables), we must convert these scalar values into objects for our logic in JavaScript to work and back from objects to scalar values for storage.
