Model View Controller

Definition:

MVC is an architectural paradigm for programming.The structure in your programming is built following a specific format. A way to structure our application. Some of the most software frameworks use the Model View Controller approach. For example Rails, Django, Laravel, CakePHP and IOS.

Model => Stores Data

Interacting with your data; Adding or retrieving items , processing data. The model only contacts the controller.

View => Is what the user sees.

Displaying to the user; HTML/CSS. The view only listens to the controller.

Controller => Interacts with the user and database.

Processes GET/POST/UPDATE/DELETE. It handles the logic. Is the man in charge as a middleman.
